# Code of Conduct

Be decent. Don't harass, don't discriminate, don't be abusive. Attack the idea, not the person. Assume good faith until demonstrated otherwise.

Report conduct concerns to harold@kanousei.com. The maintainers may remove comments, commits, code, issues, and contributors that violate this standard.

This project follows the spirit of the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) without adopting it verbatim.
